Installation

To add this extension to your project, use the relevant command in your Quarkus project directory:

Quarkus CLI

quarkus ext add org.apache.camel.quarkus:camel-quarkus-support-consul-client

Maven

./mvnw quarkus:add-extension -Dextensions="org.apache.camel.quarkus:camel-quarkus-support-consul-client"

Gradle

./gradlew addExtension --extensions="org.apache.camel.quarkus:camel-quarkus-support-consul-client"
