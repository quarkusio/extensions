Installation

To add this extension to your project, use the relevant command in your Quarkus project directory:

Quarkus CLI

quarkus ext add org.apache.camel.quarkus:camel-quarkus-support-debezium

Maven

./mvnw quarkus:add-extension -Dextensions="org.apache.camel.quarkus:camel-quarkus-support-debezium"

Gradle

./gradlew addExtension --extensions="org.apache.camel.quarkus:camel-quarkus-support-debezium"
