Installation

To add this extension to your project, use the relevant command in your Quarkus project directory:

Quarkus CLI

quarkus ext add io.quarkus:quarkus-resteasy-reactive-jackson-common

Maven

./mvnw quarkus:add-extension -Dextensions="io.quarkus:quarkus-resteasy-reactive-jackson-common"

Gradle

./gradlew addExtension --extensions="io.quarkus:quarkus-resteasy-reactive-jackson-common"
