Installation

To add this extension to your project, use the relevant command in your Quarkus project directory:

Quarkus CLI

quarkus ext add io.quarkus:quarkus-resteasy-links

Maven

./mvnw quarkus:add-extension -Dextensions="io.quarkus:quarkus-resteasy-links"

Gradle

./gradlew addExtension --extensions="io.quarkus:quarkus-resteasy-links"
